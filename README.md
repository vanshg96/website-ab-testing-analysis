# website-ab-testing-analysis
# Website A/B Testing Simulation

This project simulates an A/B testing experiment to compare two website variants.

## Features

- Simulates website visitors
- Calculates conversion rates
- Performs statistical hypothesis testing
- Uses Z-test to determine significance
- Data visualization using Python

## Technologies Used

- Python
- NumPy
- Pandas
- SciPy
- Statsmodels
- Matplotlib
- Seaborn

## Project File

website_A_B_Testing.ipynb contains the full experiment simulation and analysis.
